## Pioneer Space Simulator

Pioneer is a space adventure game set in our galaxy at the turn of the 31st century.

This snap is a community-maintained unofficial build of [Pioneer Space Simulator](https://github.com/pioneerspacesim/pioneer).

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/pioneer)

## Install

Install on any [supported Linux](https://snapcraft.io/docs/installing-snapd) distribution:

```
snap install pioneer
```
